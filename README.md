## 🏗️ Tech Stack

### Frontend

- Next.js
- Typescript
- Redux
- React Material UI
- Firebase

### Backend

- Node.js & Express
- Typescript
- RESTful API
- Firebase Admin

## ⚙️ Getting Started

### Prerequisites

- Node.js (v20+)
- npm

### Installation

#### Frontend

1. `cd apps/frontend-repo`
2. `npm install`

#### Backend

1. `cd apps/backend-repo`
2. `npm install`

#### Running Apps

1. in root project typing command `npm run dev`
2. frontend url `http://localhost:3000`
3. backend emulator url `http://127.0.0.1:4000`
